## AI Agents Incubator

We build AI agents, autonomous workflows, applied tools, and learning systems that are meant to be used in real work, not just shown in demos. This organization is part incubator, part lab, part course ecosystem, and part operating space for experiments that turn AI into usable systems.

---

### What We Focus On

- Designing autonomous AI agents and long-horizon workflows
- Building Claude Code and AI-assisted development systems around specs, routing, permissions, and memory
- Shipping applied products with Supabase, RAG, authentication, tokens, and workflow automation
- Creating tools for transcript processing, editorial workflows, and content production
- Developing learning materials, quizzes, mentorship tools, and course infrastructure

## Repository Map

*Some repositories below are private incubator workspaces and are therefore not publicly accessible.*

- [.github](https://github.com/AI-agents-incubator/.github) — Organization profile repository and shared metadata for the AI Agents Incubator account
- [supabase-wordpress](https://github.com/AI-agents-incubator/supabase-wordpress) — WordPress plugin for Supabase Auth integration with OAuth, Magic Link, JWT verification, and WordPress user sync
- [essays](https://github.com/AI-agents-incubator/essays) — Technical essays, notes, and long-form documents on vibe coding, AI-assisted development, and agent workflows
- [powereditor](https://github.com/AI-agents-incubator/powereditor) — Local transcript-cleanup utility for long spoken recordings that preserves meaning while removing recognition noise
- [n8n-pilot](https://github.com/AI-agents-incubator/n8n-pilot) — AI-powered Chrome extension for managing n8n workflows with Claude in a fully local setup
- [news](https://github.com/AI-agents-incubator/news) — Semi-automated pipeline for building authored news digests from research and LLM commentary
- [process_transcript](https://github.com/AI-agents-incubator/process_transcript) — Transcript post-processing pipeline for MacWhisper outputs with watcher automation, prompt-driven cleanup, and SRT reconstruction
- [utilities](https://github.com/AI-agents-incubator/utilities) — Utility scripts for conversion and workflow automation, including HTML and Markdown processing tools
- [course_14day](https://github.com/AI-agents-incubator/course_14day) — Private workspace for a 14-day course buildout and related experiments
- [ai-test](https://github.com/AI-agents-incubator/ai-test) — AI knowledge assessment quiz platform built with React, Vite, Tailwind, and Supabase
- [html-games](https://github.com/AI-agents-incubator/html-games) — Private HTML game experiments and browser-based gameplay prototypes
- [chatRAG](https://github.com/AI-agents-incubator/chatRAG) — RAG-enabled chat application with OpenAI Assistants, vector search, and multi-persona workflows
- [ChatOpenAIIntegration](https://github.com/AI-agents-incubator/ChatOpenAIIntegration) — ChatGPT-style frontend with Supabase auth, chat history, personas, and OpenAI integration
- [Subtutle](https://github.com/AI-agents-incubator/Subtutle) — Monorepo with multiple React apps, backend services, subtitle tooling, and assorted admin and game experiments
- [Ai_agents_course](https://github.com/AI-agents-incubator/Ai_agents_course) — Downloadable materials and course assets for the AI Agents course
- [AIMentor](https://github.com/AI-agents-incubator/AIMentor) — AI-powered mentor for coding interviews, guided walkthroughs, and structured feedback
- [git13](https://github.com/AI-agents-incubator/git13) — Minimal TypeScript sandbox for web and AI experiments, small services, and utility testing
- [AdminSupabase1.0](https://github.com/AI-agents-incubator/AdminSupabase1.0) — React and Supabase admin dashboard for secure project variable management
- [GameForge](https://github.com/AI-agents-incubator/GameForge) — Multiplayer web game platform with user auth, state sync, and extensible frontend and backend architecture
- [Apps_12222](https://github.com/AI-agents-incubator/Apps_12222) — Collection of small apps, utilities, prototypes, and experimental builds
- [chat-supabase](https://github.com/AI-agents-incubator/chat-supabase) — Real-time chat application with Supabase auth, rooms, and persistent messaging
- [agentdashboard2](https://github.com/AI-agents-incubator/agentdashboard2) — Production-oriented AI agent records manager with Airtable, Netlify Functions, versioning, and a full-stack UI

## Current Directions

- Long-horizon agents that can execute beyond a single chat turn
- Human-controlled AI systems built around specs, contracts, tests, and memory
- Secure product infrastructure with auth, RLS, permissions, and verification layers
- Applied automation for research, content production, and operational workflows
- Educational products that help non-technical people use AI effectively without surrendering judgment
